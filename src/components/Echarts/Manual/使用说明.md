# ECharts 组件库使用说明

## 概述

这是一个基于 Vue 3 和 ECharts 的图表组件库，提供了 16 种常用的图表类型，包括柱形图、折线图、饼图、散点图等。所有组件都基于统一的 `BaseChart` 组件构建，具有一致的 API 和事件处理机制。

## 安装依赖

确保项目中已安装 ECharts：

```bash
npm install echarts
# 或
yarn add echarts
```

## 快速开始

### 1. 导入组件

```javascript
// 导入单个组件
import { BarChart } from '@/components/Echarts'

// 导入多个组件
import { BarChart, LineChart, PieChart } from '@/components/Echarts'

// 导入所有组件
import * as ECharts from '@/components/Echarts'
```

### 2. 基础使用

```vue
<template>
  <div>
    <!-- 柱形图 -->
    <BarChart
      :data="chartData"
      :x-axis="xAxisData"
      title="销售数据"
      height="300px"
      @chart-click="handleClick"
    />
  </div>
</template>

<script setup>
import { ref } from 'vue'
import { BarChart } from '@/components/Echarts'

const chartData = ref([120, 200, 150, 80, 70, 110])
const xAxisData = ref(['1月', '2月', '3月', '4月', '5月', '6月'])

const handleClick = (params) => {
  console.log('图表点击:', params)
}
</script>
```

## 组件列表

### 1. BarChart - 柱形图
用于展示分类数据的对比。

**主要属性：**
- `data`: 数据数组
- `xAxis`: X轴标签数组
- `title`: 图表标题
- `subtitle`: 图表副标题
- `height`: 图表高度
- `width`: 图表宽度
- `seriesName`: 系列名称

### 2. LineChart - 折线图
用于展示数据趋势变化。

**主要属性：**
- `data`: 数据数组
- `xAxis`: X轴标签数组
- `smooth`: 是否平滑曲线
- `showArea`: 是否显示面积
- `title`: 图表标题
- `subtitle`: 图表副标题

### 3. PieChart - 饼图
用于展示部分与整体的关系。

**主要属性：**
- `data`: 数据数组，格式为 `[{name, value}]`
- `pieType`: 饼图类型 ('pie' | 'donut')
- `radius`: 半径配置
- `showLabel`: 是否显示标签
- `title`: 图表标题

### 4. AreaChart - 面积图
用于展示数据趋势和面积填充。

**主要属性：**
- `data`: 数据数组
- `xAxis`: X轴标签数组
- `stack`: 是否堆叠
- `title`: 图表标题

### 5. ScatterChart - 散点图
用于展示两个变量之间的关系。

**主要属性：**
- `data`: 数据数组，格式为 `[[x, y]]`
- `xName`: X轴名称
- `yName`: Y轴名称
- `title`: 图表标题

### 6. RadarChart - 雷达图
用于多维度数据对比。

**主要属性：**
- `data`: 数据数组
- `indicators`: 指标配置数组
- `title`: 图表标题

### 7. GaugeChart - 仪表盘
用于展示进度或指标值。

**主要属性：**
- `value`: 当前值
- `min`: 最小值
- `max`: 最大值
- `title`: 图表标题

### 8. FunnelChart - 漏斗图
用于展示转化流程。

**主要属性：**
- `data`: 数据数组，格式为 `[{name, value}]`
- `title`: 图表标题

### 9. HeatmapChart - 热力图
用于展示二维数据的密度分布。

**主要属性：**
- `data`: 数据数组，格式为 `[[x, y, value]]`
- `xAxis`: X轴标签
- `yAxis`: Y轴标签
- `title`: 图表标题

### 10. TreeChart - 树图
用于展示层级结构。

**主要属性：**
- `data`: 树形数据
- `layout`: 布局方式
- `title`: 图表标题

### 11. SunburstChart - 旭日图
用于展示多层级数据的占比。

**主要属性：**
- `data`: 层级数据
- `title`: 图表标题

### 12. TreemapChart - 矩形树图
用于展示层级数据的面积占比。

**主要属性：**
- `data`: 数据数组
- `title`: 图表标题

### 13. CandlestickChart - 蜡烛图
用于展示股票等金融数据。

**主要属性：**
- `data`: K线数据，格式为 `[[open, close, low, high]]`
- `title`: 图表标题

### 14. MapChart - 地图
用于展示地理数据。

**主要属性：**
- `data`: 地图数据
- `mapName`: 地图名称
- `title`: 图表标题

### 15. LiquidChart - 水波图
用于展示百分比或进度。

**主要属性：**
- `value`: 当前值（0-1之间）
- `title`: 图表标题

### 16. WordCloudChart - 词云图
用于展示文本数据的频率。

**主要属性：**
- `data`: 词云数据，格式为 `[{name, value}]`
- `title`: 图表标题

## 通用属性

所有组件都支持以下通用属性：

### 基础属性
- `width`: 图表宽度（默认：'100%'）
- `height`: 图表高度（默认：'400px'）
- `title`: 图表标题
- `subtitle`: 图表副标题
- `theme`: 主题配置

### 响应式属性
- `autoResize`: 是否自动调整大小（默认：true）
- `renderer`: 渲染器类型（'canvas' | 'svg'）

## 事件处理

所有组件都支持以下事件：

### 基础事件
- `chart-ready`: 图表渲染完成
- `chart-click`: 图表点击事件
- `chart-dblclick`: 图表双击事件
- `chart-mousedown`: 鼠标按下事件
- `chart-mouseup`: 鼠标释放事件
- `chart-mousemove`: 鼠标移动事件
- `chart-mouseover`: 鼠标悬停事件
- `chart-mouseout`: 鼠标离开事件

### 事件处理示例

```vue
<template>
  <BarChart
    :data="data"
    :x-axis="xAxis"
    @chart-click="handleClick"
    @chart-ready="handleReady"
  />
</template>

<script setup>
const handleClick = (params) => {
  console.log('点击的数据:', params.data)
  console.log('点击的系列:', params.seriesName)
  console.log('点击的索引:', params.dataIndex)
}

const handleReady = (chart) => {
  console.log('图表已准备就绪:', chart)
}
</script>
```

## 数据格式

### 1. 数组格式
适用于柱形图、折线图、面积图等：
```javascript
const data = [120, 200, 150, 80, 70, 110]
```

### 2. 对象数组格式
适用于饼图、漏斗图等：
```javascript
const data = [
  { name: '苹果', value: 35 },
  { name: '三星', value: 25 },
  { name: '华为', value: 20 }
]
```

### 3. 二维数组格式
适用于散点图：
```javascript
const data = [
  [170, 65], [175, 70], [160, 50], [180, 80]
]
```

### 4. 树形数据格式
适用于树图、旭日图：
```javascript
const data = {
  name: '根节点',
  children: [
    {
      name: '子节点1',
      value: 10,
      children: [
        { name: '子节点1-1', value: 5 },
        { name: '子节点1-2', value: 5 }
      ]
    }
  ]
}
```

## 主题配置

### 内置主题
```javascript
import { BarChart } from '@/components/Echarts'

// 使用内置主题
<BarChart :theme="'dark'" :data="data" />
```

### 自定义主题
```javascript
const customTheme = {
  color: ['#5470c6', '#91cc75', '#fac858'],
  backgroundColor: '#ffffff',
  textStyle: {
    color: '#333333'
  }
}

<BarChart :theme="customTheme" :data="data" />
```

## 最佳实践

### 1. 性能优化
- 对于大数据量图表，考虑使用数据采样
- 合理设置 `autoResize` 属性
- 在组件销毁时及时清理事件监听

### 2. 响应式设计
- 使用相对单位（如百分比）设置宽度
- 启用 `autoResize` 属性
- 监听窗口大小变化

### 3. 数据更新
- 使用响应式数据（ref/reactive）
- 避免频繁更新大量数据
- 考虑使用防抖处理

### 4. 错误处理
```javascript
const handleChartError = (error) => {
  console.error('图表渲染错误:', error)
  // 显示错误提示或降级处理
}
```

## 常见问题

### Q: 图表不显示怎么办？
A: 检查以下几点：
1. 确保容器有明确的宽高
2. 检查数据格式是否正确
3. 确认 ECharts 依赖已正确安装

### Q: 图表大小不正确？
A: 尝试以下解决方案：
1. 设置明确的 `width` 和 `height` 属性
2. 启用 `autoResize` 属性
3. 手动调用 `resizeChart()` 方法

### Q: 如何自定义样式？
A: 可以通过以下方式：
1. 使用 `theme` 属性设置主题
2. 通过 CSS 自定义容器样式
3. 使用 ECharts 的样式配置选项
